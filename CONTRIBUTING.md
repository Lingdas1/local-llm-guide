# Contributing

Contributions are welcome! Here's how you can help:

- **Hardware benchmarks**: Run the benchmark script and submit your results
- **New model guides**: Write a guide for a model not yet covered
- **Bug fixes**: Fix errors in existing guides or scripts
- **Translations**: Translate this guide to other languages

## Guidelines

1. Keep it beginner-friendly — assume the reader has a consumer GPU or even just a CPU
2. Test your scripts before submitting
3. Use clear, plain English

Thanks for contributing! 🦙
